# Hungry House Hub - AI-Powered Restaurant Management System

This is a modern, AI-powered restaurant management application built with Next.js and Firebase in Firebase Studio.

## Features

- **Menu Management**: Admins can easily add, edit, and delete menu items and categories.
- **Order Entry**: Staff can quickly take customer orders through an intuitive interface.
- **Invoice Generation**: Automatically generate and display an invoice after an order is placed.
- **AI-Powered Features (Coming Soon!)**:
    - **AI-Generated Descriptions**: Automatically generate mouth-watering descriptions for new menu items using AI.
    - **AI-Generated Images**: Create unique, professional-quality images for dishes to replace placeholders.
    - **AI Price Suggestions**: Get intelligent price suggestions for new items.

## Tech Stack

- **Framework**: Next.js (React)
- **Styling**: Tailwind CSS, ShadCN UI
- **AI**: Google Gemini (via Genkit)
- **Database & Auth**: Firebase (Firestore, Firebase Auth)

## Getting Started

To run the development server, use the following command:

```bash
npm run dev
```

You can then view the application at [http://localhost:9002](http://localhost:9002).
