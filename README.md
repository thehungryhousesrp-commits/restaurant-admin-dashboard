# Hungry House Hub - Restaurant Management System

This is a modern, AI-powered restaurant management application built with Next.js and Firebase in Firebase Studio.

## Features

- **Menu Management**: Admins can easily add, edit, and delete menu items and categories.
- **Order Entry**: Staff can quickly take customer orders through an intuitive interface.
- **Invoice Generation**: Automatically generate and display an invoice after an order is placed.
- **AI-Powered (Coming Soon!)**: Future features will include AI-generated menu descriptions and images.

## Getting Started

To run the development server, use the following command:

```bash
npm run dev
```

You can then view the application at [http://localhost:9002](http://localhost:9002).
